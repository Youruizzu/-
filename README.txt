Метод AppendStringArray:
Увеличивает размер массива на 1 
и добавляет новый элемент в конец массива.

Метод GetSmallElementsFromArray:
Делает новый массив из элементов данного массива,
которые имеют размер равный или меньший трёх.

Метод PrintArray:
Выводит данный массив на экран.
Массив обрамлён "[" и "]", а элементы перечислены через запятую.
